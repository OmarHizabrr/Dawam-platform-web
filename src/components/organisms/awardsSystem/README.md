# نظام الجوائز والميداليات - Awards System

## نظرة عامة

نظام متطور لتقييم أداء الموظفين وتحفيزهم من خلال منح الكؤوس والميداليات بناءً على أدائهم في الحضور والانضباط.

## فلسفة النظام

### 🏆 الكؤوس - تقييم الانضباط الشهري

يتم منح الكؤوس بناءً على نسبة ساعات العمل الفعلية خلال الشهر:

#### 🥉 الكأس البرونزي

- **المعيار**: 80-89% من ساعات العمل الفعلية
- **اللون**: برونزي (#CD7F32)
- **الهدف**: تشجيع الموظفين على تحسين أدائهم

#### 🥈 الكأس الفضي

- **المعيار**: 90-95% من ساعات العمل الفعلية
- **اللون**: فضي (#C0C0C0)
- **الهدف**: تقدير الأداء الجيد

#### 🥇 الكأس الذهبي

- **المعيار**: 95-99% من ساعات العمل الفعلية
- **اللون**: ذهبي (#FFD700)
- **الهدف**: تكريم الأداء المتميز

#### 💎 الكأس الماسي

- **المعيار**: 100% دوام مثالي خلال الشهر
- **اللون**: تدرج أزرق ماسي
- **الهدف**: تكريم الكمال في الانضباط

### 🏅 الميداليات - التنافس اليومي

يتم منح الميداليات بناءً على ترتيب الحضور اليومي:

#### 🥇 الميدالية الذهبية

- **المعيار**: أول الحاضرين في اليوم
- **العدد المعروض**: إجمالي عدد المرات التي حصل فيها على المركز الأول

#### 🥈 الميدالية الفضية

- **المعيار**: ثاني الحاضرين في اليوم
- **العدد المعروض**: إجمالي عدد المرات التي حصل فيها على المركز الثاني

#### 🥉 الميدالية البرونزية

- **المعيار**: ثالث الحاضرين في اليوم
- **العدد المعروض**: إجمالي عدد المرات التي حصل فيها على المركز الثالث

### ⭐ الكأس المميز - الإنجاز الاستثنائي

- **المعيار**: أطول فترة حضور متتالية دون أي غيابات
- **الرمز**: كأس مع نجمة ⭐
- **العدد المعروض**: عدد الأيام المتتالية (بدون احتساب أيام الجمعة)

## الحسابات التقنية

### حساب نسبة ساعات العمل

```javascript
النسبة المئوية = (إجمالي الساعات الفعلية / إجمالي الساعات المطلوبة) × 100
```

### حساب الفترة المتتالية

- يتم تجاهل أيام الجمعة والعطل الرسمية
- يتم حساب الأيام المتتالية التي تحتوي على حضور فعلي
- تنقطع السلسلة عند أول يوم غياب

## المميزات التقنية

### 🎨 التأثيرات البصرية

- تأثيرات التوهج للكؤوس المختلفة
- حركات النبض للميداليات
- تأثير الدوران للكأس المميز عند التمرير

### 📱 الاستجابة للشاشات

- تصميم متجاوب يعمل على جميع أحجام الشاشات
- تقليل حجم الأيقونات على الشاشات الصغيرة
- إعادة ترتيب العناصر حسب حجم الشاشة

### 💡 تلميحات الأدوات

- تلميحات توضيحية تظهر عند التمرير
- شرح مفصل لكل نوع من أنواع الجوائز
- معايير الحصول على كل جائزة

## الاستخدام

### التكامل مع النظام

```jsx
import AwardsSystem from "./components/organisms/awardsSystem";

<AwardsSystem
  userData={user}
  attendanceData={attendanceData}
  setting={setting}
/>;
```

### البيانات المطلوبة

- `userData`: بيانات الموظف الأساسية
- `attendanceData`: سجل الحضور والانصراف
- `setting`: إعدادات النظام

## التطوير المستقبلي

### مقترحات التحسين

1. **نظام النقاط**: تحويل الجوائز إلى نقاط قابلة للتجميع
2. **المكافآت**: ربط الجوائز بمكافآت مالية أو معنوية
3. **التقارير**: إنشاء تقارير تفصيلية عن أداء الموظفين
4. **المقارنات**: مقارنة الأداء بين الموظفين والأقسام
5. **الإشعارات**: تنبيهات عند الحصول على جوائز جديدة

### API المطلوبة

```javascript
// للحصول على ترتيب الحضور اليومي
GET / daily - attendance - ranking / { date };

// للحصول على إحصائيات الموظف الشهرية
GET / employee - monthly - stats / { user_id } / { start_date } / { end_date };
```
